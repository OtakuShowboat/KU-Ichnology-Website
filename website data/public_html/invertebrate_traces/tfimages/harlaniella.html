Harlaniella: Sokolov, 1972

DESCRIPTION: Smooth walled, rope-like burrow that can be branched or unbranched; form parallel to bedding plane; 1-4 mm in diameter.

BEHAVIOR(S): Fodinichnia

ENVIRONMENTAL SETTINGS: Marine

POSSIBLE TRACEMAKERS: Annelid worms

GEOLOGIC RANGE: Ediacaran - Cambrian

ADDITIONAL REFERENCES: Signor, 1994

REMARKS: Description from Signor, 1994. Jensen (2003) thinks Harlaniella should be considered nomen nudum, since some spiral segments turn parallel to the burrow structure suggesting the burrow does not spiral.
